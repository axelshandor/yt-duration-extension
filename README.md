## üöÄ Features

- **Instant Calculation:** Displays total hours, minutes, and seconds directly on the playlist page.
- **Smart "Scroll & Update" Button:** YouTube only loads a few videos at a time (lazy loading). Our unique feature automatically scrolls through long playlists to capture every single video and update the total time accurately.
- **Native Integration:** Designed to look and feel like a part of the YouTube interface.
- **Privacy Focused:** No data collection, no tracking. Calculations happen entirely in your browser.

### For Developers (Manual Install)
1. Clone this repository or download the ZIP file.
2. Extract the files to a local folder.
3. Open Google Chrome and navigate to `chrome://extensions/`.
4. Enable **"Developer mode"** (top right corner).
5. Click **"Load unpacked"** and select the folder containing the extension files.

## üìñ How to Use

1. Open any **YouTube Playlist** (URL must start with `https://www.youtube.com/playlist...`).
2. The **Total Duration** will appear near the playlist metadata.
3. For long playlists, click the **"Scroll & Update"** button to load all videos and get the full duration.
